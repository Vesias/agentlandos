## 🏗️ TECHNICAL ARCHITECTURE

- **Frontend**: Next.js 14 + TypeScript
- **Backend**: Python FastAPI + PostgreSQL
- **KI-Brain**: DeepSeek-R1-0528 (Reasoning)
